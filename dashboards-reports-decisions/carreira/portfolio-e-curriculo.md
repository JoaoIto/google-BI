# 📄 Portfólio e Currículo em BI

Seu currículo diz o que você sabe fazer; seu portfólio **prova** o que você já fez.

---

## 🛠️ Onde Hospedar seu Trabalho
1.  **Tableau Public**: O padrão ouro para visualizações interativas.
2.  **GitHub / Kaggle**: Excelente para compartilhar scripts de limpeza de dados (SQL/Python) e datasets.
3.  **Site Personalizado**: Plataformas como Wix, Squarespace ou GitHub Pages para criar uma narrativa visual completa.

---

## 📝 O que um Projeto de Portfólio deve conter:
- **Título e Resumo**: O que é o projeto em uma frase.
- **Capturas de Tela**: Imagens do painel final.
- **Documentação de Processo**:
    - Qual era o problema?
    - Quais dados foram usados?
    - Como foram limpos/transformados?
    - Quais foram os principais insights?
- **Auto-Reflexão**: O que você aprendeu e como evoluiria o projeto.

---

## 🔒 Privacidade e Ética
**IMPORTANTE**: Nunca compartilhe dados privados de empregadores anteriores.
- Se os dados forem sensíveis, compartilhe apenas um **resumo do processo** e os **resultados de alto nível**, sem mostrar imagens ou dados brutos.
- Use datasets públicos (Kaggle, BigQuery) para criar projetos autorais que você possa exibir livremente.

---

## 📑 O Currículo de BI
- Destaque ferramentas (Tableau, SQL, Looker, BigQuery).
- Use verbos de ação: "Desenvolvi", "Otimizei", "Liderei", "Reduzi o tempo de...".
- Inclua o link para o seu portfólio no topo, logo abaixo dos seus contatos.
