# Desafio: Criação de Conjunto de Slides (Atividade 4.1)

Este documento registra as respostas para a atividade inicial de criação de apresentações de BI.

---

### 1. Componentes de uma Apresentação de BI
**Pergunta:** Quais componentes devem ser incluídos em uma apresentação de slides de BI para um supervisor ou parte interessada?
*   **Respostas Selecionadas:**
    *   Um slide de introdução.
    *   Um resumo das perguntas comerciais que você respondeu.
    *   Uma descrição de seus métodos para criar seu projeto.
    *   Um resumo das principais percepções sobre os dados.
*   **Justificativa:** Estes quatro pilares garantem que o público entenda quem está falando, qual problema está sendo resolvido, como a solução foi construída e quais são as conclusões acionáveis.

### 2. Localização do Pipeline
**Pergunta:** Em qual slide de sua apresentação você poderia incluir uma descrição de como criou seu pipeline?
*   **Resposta:** **Métodos**
*   **Justificativa:** O slide de métodos é o local apropriado para detalhar os aspectos técnicos, processos de dados e ferramentas utilizadas.

### 3. Foco do Slide de Insights
**Pergunta:** Em seu slide de insights principais, você deve procurar fornecer ________.
*   **Resposta:** **respostas diretas às perguntas da empresa**
*   **Justificativa:** O valor do BI reside na utilidade prática. O stakeholder busca respostas para seus problemas de negócio, não apenas curiosidades sobre os dados.
