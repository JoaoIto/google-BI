# Desafio: Criação de Gráficos de Tráfego (MnDOT)

Este documento registra as respostas para a atividade de criação de gráficos para o Departamento de Transportes de Minnesota.

---

### 1. Conclusão da Atividade
**Pergunta:** Você concluiu esta atividade?
*   **Resposta:** Sim. (A documentação técnica no README serve como base lógica).

### 2. Escolha de Gráfico para Feriados
**Pergunta:** Qual dos seguintes tipos de gráfico ilustraria com mais eficácia os feriados com maior tráfego?
*   **Resposta:** **Gráfico de círculos (bolhas)**
*   **Justificativa:** Em contextos de BI, o gráfico de bolhas é frequentemente utilizado para comparar o volume de diferentes categorias (como nomes de feriados) de forma visualmente atraente, onde o tamanho do círculo representa a magnitude do tráfego.

### 3. Acessibilidade
**Pergunta:** Qual dos seguintes elementos melhora a acessibilidade de seus gráficos?
*   **Resposta:** **Legendas**
*   **Justificativa:** O Tableau possui um recurso de "Legendas" (Captions) que gera descrições automáticas dos detalhes do gráfico, essencial para leitores de tela e compreensão contextual.
