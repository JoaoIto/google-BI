# Desafio: Iteração e Refinamento do Painel (MnDOT)

Este documento registra as respostas para a atividade de iteração baseada no feedback do stakeholder Camila.

---

### 1. Conclusão da Atividade
**Pergunta:** Você concluiu esta atividade?
*   **Resposta:** Sim.

### 2. Objetos e Botões no Dashboard
**Pergunta:** Quais são alguns dos tipos de botões que você pode adicionar ao seu painel?
*   **Respostas Selecionadas:**
    *   **Imagens**
    *   **Navegação**
    *   **Baixar** (Download)
*   **Justificativa:** O menu "Objetos" do Tableau permite a inclusão de diversos artefatos interativos. O botão de download foi especificamente solicitado pela Camila para facilitar a exportação de dados.

### 3. Utilidade dos Blocos de Filtro
**Pergunta:** Como a adição de um bloco à interface do painel que permite que os próprios usuários alterem os filtros atende às solicitações das partes interessadas?
*   **Respostas Selecionadas:**
    *   **Incentiva os usuários a visualizar diferentes escalas de tempo.**
    *   **Ajuda os usuários que não estão familiarizados com o Tableau a aplicar filtros.**
*   **Justificativa:** Menus suspensos e blocos de filtros externos são intuitivos para usuários leigos, permitindo que eles explorem os dados (mudando de meses para horas, por exemplo) sem precisar conhecer as ferramentas internas do Tableau.
