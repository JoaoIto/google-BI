# Desafio: Montagem e Organização do Painel (MnDOT)

Este documento registra as respostas para a atividade de organização do painel para o Departamento de Transportes de Minnesota, focando em hierarquia visual e fluxo de informações.

---

### 1. Conclusão da Atividade
**Pergunta:** Você concluiu esta atividade?
*   **Resposta:** Sim.

### 2. Práticas Recomendadas de Design
**Pergunta:** Quais práticas recomendadas podem ajudá-lo a criar seu painel?
*   **Respostas Selecionadas:**
    *   Organize as legendas junto com os gráficos.
    *   Use o tamanho, a cor e outros elementos de design para criar um fluxo lógico.
    *   Primeiro, coloque o gráfico que deseja enfatizar em seu painel.
*   **Justificativa:** O design é um processo iterativo. Embora o mockup seja um guia, ajustes são necessários durante a construção para garantir que o elemento principal (neste caso, o volume anual) receba o maior foco visual.

### 3. Organização Lógica
**Pergunta:** Quais são algumas maneiras lógicas de organizar os gráficos em seu painel?
*   **Respostas Selecionadas:**
    *   Agrupe os gráficos por tópico.
    *   Organize os gráficos do mais importante para o menos importante.
    *   Use cores semelhantes.
*   **Justificativa:** Organizar por importância segue o padrão de leitura em "F" ou "Z" (colocando o insight mais crítico no topo). Manter cores semelhantes (paleta consistente) ajuda na unidade visual e reduz a carga cognitiva do usuário. Preencher todo o espaço não é recomendado, pois o espaço negativo é vital para a legibilidade.
