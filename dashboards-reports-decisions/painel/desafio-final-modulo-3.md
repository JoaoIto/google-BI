# 🏆 Desafio Final: Módulo 3 - Dashboards e Relatórios

Este documento consolida as respostas e fundamentos técnicos para a avaliação final do Módulo 3 do curso de BI do Google.

---

### 1. As Quatro Perguntas Fundamentais do Design
**Pergunta:** Além de "Quais partes do conjunto de dados permitirão atender aos requisitos da empresa?", quais são as outras três perguntas fundamentais?
*   **Respostas Selecionadas:**
    *   Quais tipos de gráficos e formas de codificação serão mais eficazes?
    *   Quantos gráficos são necessários?
    *   Quais medidas e dimensões serão representadas em cada gráfico?
*   **Fundamento**: O design de BI começa com a seleção correta dos dados (medidas/dimensões), passa pela escolha da codificação visual mais eficiente e termina no equilíbrio quantitativo para evitar a saturação de informação.

### 2. Riscos de Excesso de Visualizações
**Pergunta:** Quais são os riscos de incluir várias visualizações em um painel?
*   **Respostas Selecionadas:**
    *   Diminuir a velocidade do painel de controle (Performance).
    *   Sobrecarregar o usuário (Carga Cognitiva).
*   **Fundamento**: Painéis muito densos aumentam o tempo de carregamento e dificultam a identificação rápida de insights, indo contra o princípio de simplicidade do BI.

### 3. Contexto em Monitoramento de Longo Prazo
**Pergunta:** Painéis maiores provavelmente exigirão mais contexto para tornar os dados ________ para os usuários.
*   **Resposta:** **compreensível**
*   **Fundamento**: Sem contexto (metas, históricos, referências), os dados de longo prazo tornam-se apenas números abstratos.

### 4. Importância dos Casos de Uso (Live Monitoring)
**Pergunta:** Por quais motivos é importante estabelecer casos de uso para diferentes visualizações?
*   **Respostas Selecionadas:**
    *   Para simplificar o processo de acesso a dados úteis.
    *   Para evitar que os usuários tenham que interpretar dados irrelevantes para eles.
    *   Oferece a diferentes usuários a capacidade de detalhar (drill-down) as informações de que precisam.
*   **Fundamento**: Casos de uso garantem que a visualização seja centrada no usuário, entregando o dado certo para a pessoa certa no momento certo.

### 5. Evolução vs. Instantâneos
**Pergunta:** É útil criar um rastreamento que permita aos usuários acompanhar os dados ________, em vez de instantâneos isolados.
*   **Resposta:** **evolução**
*   **Fundamento**: O monitoramento de BI foca na tendência e na progressão ao longo do tempo para prever comportamentos futuros.

### 6. Benefício das Comparações
**Pergunta:** A implementação de ________ é benéfica porque ajuda os usuários a entender por que os dados do painel são úteis.
*   **Resposta:** **comparações**
*   **Fundamento**: Um dado só ganha significado quando comparado a um benchmark, período anterior ou meta pré-estabelecida.
