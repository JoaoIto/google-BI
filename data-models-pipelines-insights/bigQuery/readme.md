# 🔍 BigQuery - Estudos e Práticas

Esta pasta centraliza as atividades e estudos relacionados ao **Google BigQuery** dentro do módulo de Modelagem de Dados, Pipelines e Insights.

## 📂 Conteúdo

### 1️⃣ [Atividade 1: Tabela de Destino](./activity-1-destination-table/README.md)
Foco na extração de dados públicos (árvores de São Francisco) e criação de tabelas de destino para análise de BI.
- **Conceitos:** Extração, SQL (SELECT, GROUP BY), CSV local.

### 2️⃣ [Atividade 2: Fusão de Dados e Sandbox](./activity-2-fusion-and-sandbox/README.md)
Estudo de caso sobre a importância da fusão de dados de múltiplas fontes e guia prático para uso do BigQuery sem custo.
- **Conceitos:** Data Fusion, Stakeholders, Sandbox Environment.

### 3️⃣ [Atividade 3: Unificar Dados com Tabelas de Destino](./activity-3-create-destination/README.md)
Referência teórica sobre tipos de extração de dados e unificação em tabelas de destino, acompanhada dos dados exportados (CSV).
- **Conceitos:** ETL, Notificação vs Incremental vs Total, Exportação CSV.

### 4️⃣ [Atividade 4: Estudo de Caso Wayfair](./activity-4-wayfair-case-study/README.md)
Estudo sobre a colaboração com stakeholders para o design de pipelines eficientes e escaláveis em larga escala.
- **Conceitos:** Stakeholder Management, Pricing Catalog, Scalability.

### 5️⃣ [Atividade 5: Tendências Culturais com Google Books Ngrams](./activity-5-cultural-trends-ngrams/README.md)
Exploração de um dataset massivo (Petabytes) para analisar o uso de termos e conceitos ao longo dos anos.
- **Conceitos:** Big Data, Time-series analysis, Ngrams.
